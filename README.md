# HR Workforce Analytics - MySQL

Personal project where I built a small HR database in MySQL 
to practice data analysis and workforce reporting.

## What I built

4 connected tables: departments, employees, salaries and attendance.

## Queries I wrote

**Average Salary by Department**
Wanted to see how salaries differ across teams and practice 
JOIN across multiple tables.

**Turnover Rate per Quarter**
Calculated what % of employees left each quarter. 
Useful metric in any HR team.

**Headcount Trend**
Tracked how many active employees existed each month 
to understand workforce growth over time.

## Tools
MySQL + MySQL Workbench
